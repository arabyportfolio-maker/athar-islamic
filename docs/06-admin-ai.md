### Dashboard Home — الصفحة الرئيسية

#### KPI Cards (البطاقات الإحصائية)

| المقياس | الوصف |
|---------|-------|
| **Total Users** | إجمالي المستخدمين المسجلين |
| **Active Users Today** | المستخدمون النشطون اليوم |
| **Memorial Pages** | إجمالي صفحات الأثر |
| **Tasbeeh Today** | التسبيحات المسجلة اليوم |
| **Quran Sessions** | جلسات القراءة اليوم |
| **New Registrations** | تسجيلات جديدة (آخر 24 ساعة) |
| **Open Reports** | بلاغات مفتوحة |
| **Online Users** | المستخدمون المتصلون الآن 🔴 |

#### Charts (الرسوم البيانية)

```
Line Chart   → Daily Users (آخر 30 يوم)
Bar Chart    → Monthly Users
Map Chart    → توزيع المستخدمين حسب الدول
Pie Chart    → أنواع الأجهزة (Mobile / Desktop / Tablet)
Funnel Chart → Traffic Sources
Bar Chart    → أكثر الميزات استخدامًا
```

#### Live Feed — التغذية الحية

```
🟢 مستخدم جديد: أحمد محمد — مصر
📜 صفحة أثر جديدة: "المرحوم محمد عبد الله"
📿 ختمة مكتملة في صفحة أثر
🚨 بلاغ جديد: تعليق مخالف
💬 تعليق جديد بانتظار المراجعة
```

## الجزء الثاني — AI Features

> **ملاحظة مهمة:** جميع ميزات الذكاء الاصطناعي تعمل في حدود الأمور العامة والتنظيمية فقط. لا تُصدر فتاوى أو أحكامًا شرعية. الإجابات مبنية على مصادر موثوقة.

---

### 1. Athar AI Assistant — المساعد الذكي

زر ثابت في الواجهة يمكن المستخدم من سؤاله:

```
❓ ما هو أفضل ذكر الآن؟
📖 ماذا أقرأ اليوم؟
📿 اقترح لي وردًا مناسبًا.
🕌 ماذا أفعل بعد الصلاة؟
```

---

### 2. Smart Recommendation — التوصيات الذكية

```
الوقت → التوصية

🌅 الصباح        → أذكار الصباح
🌙 المساء        → أذكار المساء
🕌 بعد الصلاة   → التسبيح (33 × 3)
📅 يوم الجمعة   → سورة الكهف
🌙 رمضان        → خطة الختمة اليومية
✨ العشر الأواخر → مضاعفة العبادة
```

---

### 3. AI Search — البحث الذكي

```
المستخدم يكتب: "دعاء الكرب"
↓
النظام يعرض: دعاء الكرب، صوته، مصدره، فوائده، وقت قراءته
```

```
المستخدم يكتب: "أذكار السفر"
↓
النظام يعرض: قائمة أذكار السفر مع الترتيب الصحيح
```

---

### 4. Smart Memorial Setup — الإعداد الذكي لصفحة الأثر

عند إنشاء صفحة جديدة، النظام يقترح الترتيب المثالي:

```
1️⃣ Biography — السيرة الذاتية
2️⃣ Quran — ختمة القرآن
3️⃣ Dua — الأدعية المهداة
4️⃣ Tasbeeh — التسبيح الجماعي
5️⃣ Comments — التعليقات والذكريات
6️⃣ Share — مشاركة الصفحة
```

---

### 5. AI Summary — التلخيص الذكي

يلخص بلغة بسيطة:
- فوائد الذكر
- شروحات المحتوى الإسلامي
- مقالات علمية بلغة مفهومة

---

### 6. Smart Notification — الإشعار الذكي

```
إذا غاب المستخدم 3+ أيام:
→ "اشتقنا إليك، لا تنس وردك اليوم 🌿"

إذا اقترب موعد صلاة:
→ "حان وقت صلاة العصر 🕌"

إذا أكمل هدفًا:
→ "أحسنت! أكملت هدف اليوم ✨"
```

---

### 7. Smart Goals — الأهداف الذكية

النظام يقترح أهدافًا تصاعدية حسب نشاط المستخدم:

```
المستخدم الجديد:
  ← 100 استغفار يوميًا

بعد أسبوع:
  ← 300 تسبيحة

بعد شهر:
  ← 500 ذكر

المستخدم المستمر:
  ← 1000 ذكر يوميًا
```

---

### 8. AI Translation — الترجمة الذكية

يترجم تلقائيًا:
- نصوص الأذكار إلى لغة المستخدم المفضلة
- واجهة التطبيق
- التعليقات في صفحات الأثر

---

### 9. Smart Accessibility — إمكانية الوصول الذكية

```
إذا كان المستخدم يكبّر الخط يدويًا كثيرًا:
→ يقترح النظام تفعيل "وضع كبار السن"

وضع كبار السن يشمل:
  ← خط أكبر تلقائيًا
  ← أزرار أكبر
  ← تباين ألوان أعلى
  ← صوت توجيهي
```

---

### 10. AI Insights — الرؤى الأسبوعية

كل أسبوع، يعرض النظام تقريرًا شخصيًا:

```
📊 تقرير أسبوعك:
  ← أكثر ذكر قلته: "سبحان الله" (1,245 مرة)
  ← أكثر وقت تتعبد فيه: بعد الفجر
  ← مدى الالتزام بأهداف الأسبوع: 86%
  ← عدد مرات فتح القرآن: 12 مرة
```

> لا يعرض التطبيق أي تقديرات للثواب أو الأجر — هذا أمر غيبي يعلمه الله وحده.

---

### 11. AI Content Moderation — مراقبة المحتوى

يفحص تلقائيًا:
- التعليقات قبل نشرها
- أسماء صفحات الأثر
- الصور المرفوعة

ويمنع أي محتوى مخالف أو مسيء تلقائيًا قبل وصوله للمشرفين.

## التوصيات النهائية للمشروع

1. **اسم الواجهة**: استخدم "صفحة الأثر" بدلًا من "Memorial" — أشمل وأعمق
2. **نظام Plugins**: صمم البنية لتقبل إضافات مستقبلية (زكاة، حج، تحفيظ)
3. **حسابات المساجد**: أضف نوع حساب خاص للمساجد والمؤسسات
4. **لا عداد حسنات**: عرض الأعمال فقط بالأرقام، لا تقدير للثواب
5. **الهوية البصرية**: شعار يرمز للأثر الممتد — دائرة تنمو أو ورقة شجرة
