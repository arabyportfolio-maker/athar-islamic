> **ملاحظة للمصمم أو AI:**  
> اللغة الأساسية عربية. كل شاشة تدعم RTL افتراضيًا.  
> لكل شاشة: Loading State (Skeleton) · Empty State · Error State · Offline State · Success State

> الألوان الأساسية: `Primary #059669` · `Secondary #D4A017` · `Background #FAFAF8`
