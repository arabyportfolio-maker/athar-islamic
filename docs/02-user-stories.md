# 02 — User Stories
# قصص المستخدمين

**Athar Islamic · Version 1.0**

---

## تعريف الأدوار

| الدور | الوصف |
|------|-------|
| **الضيف (Guest)** | يتصفح بدون تسجيل |
| **المستخدم (User)** | مسجل، يستخدم كل الميزات الأساسية |
| **Premium** | مشترك، يستخدم ميزات متقدمة |
| **Moderator** | يراجع المحتوى |
| **Admin** | يدير المنصة |

---

## Module 1 — Authentication

```
كمستخدم جديد،
  أريد أن أرى شاشة ترحيب تشرح ميزات التطبيق،
  حتى أفهم قيمة المنصة قبل التسجيل.

كمستخدم،
  أريد التسجيل بالبريد الإلكتروني أو Google أو Apple،
  حتى أتمكن من الوصول لحسابي من أي جهاز.

كمستخدم غير مسجل،
  أريد تصفح صفحات الأثر العامة والأذكار بدون تسجيل،
  حتى أتعرف على المنصة قبل الالتزام بإنشاء حساب.

كمستخدم،
  أريد اختيار لغتي المفضلة عند التسجيل،
  حتى أستخدم التطبيق بلغتي الأم.
```

---

## Module 2 — Home

```
كمستخدم،
  أريد أن أرى تحية تعتمد على اسمي والوقت الحالي،
  حتى أشعر بأن التطبيق يعرفني.

كمستخدم،
  أريد رؤية أوقات الصلاة لمدينتي تلقائيًا،
  حتى لا أحتاج لتطبيق منفصل.

كمستخدم،
  أريد الاستمرار في قراءة القرآن من حيث توقفت،
  حتى لا أضيع مكاني كل مرة.

كمستخدم،
  أريد رؤية هدفي اليومي وتقدمي الحالي في الصفحة الرئيسية،
  حتى أبقى محفوزًا طوال اليوم.

كمستخدم،
  أريد رؤية إحصائيات المجتمع الحية (عدد التسبيحات الآن)،
  حتى أشعر أنني جزء من منظومة أكبر.
```

## Module 3 — Tasbeeh

```
كمستخدم،
  أريد عداد تسبيح ضخم وسهل الضغط،
  حتى أتسبح بدون تشتيت أو صعوبة.

كمستخدم،
  أريد اختيار الذكر (سبحان الله / الحمد لله / الله أكبر / ...) قبل البدء،
  حتى أتتبع كل نوع منفصلًا.

كمستخدم،
  أريد أن أضع هدفًا للجلسة (مثل 100 تسبيحة)،
  حتى أعرف متى اكتملت.

كمستخدم،
  أريد اهتزازًا خفيفًا عند كل ضغطة (Haptic),
  حتى أتأكد من العدّ حتى عندما لا أنظر للشاشة.

كمستخدم،
  أريد رؤية سجل جلساتي اليومية والأسبوعية،
  حتى أتابع التزامي بالوقت.

كمستخدم،
  أريد رؤية أفضل يوم لي وأطول سلسلة مستمرة،
  حتى أكون محفوزًا للمحافظة عليها.
```

## Module 4 — Azkar

```
كمستخدم،
  أريد رؤية الأذكار مصنفة (صباح / مساء / صلاة / نوم / سفر)،
  حتى أجد ما يناسب وقتي بسرعة.

كمستخدم،
  أريد قراءة كل ذكر مع مصدره وترجمته وفائدته،
  حتى أقرأ بفهم وإدراك.

كمستخدم،
  أريد الاستماع لصوت الذكر،
  حتى أتعلم النطق الصحيح.

كمستخدم،
  أريد حفظ الأذكار المفضلة،
  حتى أصل إليها بسرعة دون البحث كل مرة.

كمستخدم،
  أريد مشاركة ذكر مع أصدقائي،
  حتى أنشر الخير.
```

## Module 5 — Quran

```
كمستخدم،
  أريد قراءة القرآن بخط جميل وكبير،
  حتى لا يتعب عيني أثناء القراءة.

كمستخدم،
  أريد الاستماع لقراء مشهورين بأصوات واضحة،
  حتى أستمتع بالتلاوة وأتعلم منها.

كمستخدم،
  أريد وضع علامات مرجعية على الآيات،
  حتى أرجع إليها لاحقًا.

كمستخدم،
  أريد البحث عن آية بكلمة أو عبارة،
  حتى أجد ما أريد سريعًا.

كمستخدم،
  أريد وضع ليلي (Night Mode) للقراءة المريحة،
  حتى أقرأ في الظلام بدون إجهاد.

كمستخدم،
  أريد رؤية الترجمة والتفسير تحت كل آية،
  حتى أفهم ما أقرأه.
```

## Module 6 — Legacy Hub (صفحات الأثر)

```
كمستخدم،
  أريد إنشاء صفحة أثر لشخص متوفٍّ،
  حتى يتمكن أهله وأصدقاؤه من الدعاء له والمشاركة في أعمال الخير بنية صالحة.

كمستخدم،
  أريد إنشاء صفحة أثر لنفسي (وأنا حي)،
  حتى أتتبع وردي اليومي وأشاركه مع عائلتي.

كمستخدم،
  أريد إنشاء صفحة لحملة خير (مثل: دعاء لمرضى / ختمة لأهل غزة)،
  حتى يشارك فيها أكبر عدد ممكن.

كزائر الصفحة،
  أريد المشاركة في التسبيح أو الدعاء أو قراءة القرآن داخل الصفحة،
  حتى أساهم بشيء طيب.

كمالك الصفحة،
  أريد التحكم في من يرى الصفحة (عام / خاص / عائلة)،
  حتى أحافظ على خصوصية عائلتي.

كمستخدم،
  أريد رؤية "شجرة الأثر" تنمو مع كل مشارك جديد،
  حتى أرى أثر كل شخص بشكل بصري جميل.

كمستخدم،
  أريد مشاركة الصفحة عبر رمز QR ورابط مباشر وواتساب،
  حتى تصل لأكبر عدد من الناس.
```

## Module 7 — Community

```
كمستخدم،
  أريد رؤية عدد المتسبحين الآن حول العالم،
  حتى أشعر بوحدة المسلمين وارتباطنا ببعض.

كمستخدم،
  أريد الانضمام لتحديات جماعية (مليون تسبيحة / ختمة جماعية)،
  حتى أكون جزءًا من هدف مشترك.

كمستخدم،
  أريد رؤية المتصدرين في التحديات،
  حتى تزيد حماستي للمشاركة.
```

---

## Module 8 — Profile

```
كمستخدم،
  أريد رؤية إجمالي عباداتي ومدة سلسلتي المتواصلة،
  حتى أتابع تقدمي بمرور الوقت.

كمستخدم،
  أريد فتح الإنجازات (Achievements) بشارات جميلة،
  حتى أبقى محفوزًا للاستمرار.

كمستخدم،
  أريد تغيير ثيم التطبيق (فاتح / داكن / ذهبي / زمردي)،
  حتى يناسب ذوقي وبيئة استخدامي.

كمستخدم،
  أريد تغيير لغة الواجهة بسهولة،
  حتى يستخدمه أفراد عائلتي بلغات مختلفة.
```

---

## Module 9 — Admin

```
كمشرف،
  أريد رؤية إحصائيات المنصة الحية على لوحة تحكم،
  حتى أتابع صحة المنصة وأداءها.

كمشرف،
  أريد مراجعة صفحات الأثر قبل نشرها للعموم،
  حتى أضمن محتوى لائقًا ومناسبًا.

كمشرف،
  أريد إدارة الأذكار والقرآن (إضافة / تعديل / حذف)،
  حتى أضمن صحة المحتوى الديني.

كمشرف،
  أريد مراجعة البلاغات والتعامل معها،
  حتى تبقى المنصة آمنة ونظيفة.
```
